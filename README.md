# Finance Demo App

React Native app built using Expo connected to Node.js REST Api and cloud mongoDB.

ROAD MAP FUNCTIONALITIES:

- Sing Up
- Sign In
  - User credentials
  - Google Authentication
- Sign Out
- User Profile
- Home Screen
  - Product Carousel
  - Welcome section
  - Recent transactions
- Settings Screen
  - Multi-language
  - App Version
- Transaction
  - Initial, Confirmation and Result
